# stringfuns
 
This is **stringfuns**, a small collection of JavaScript string functions that
should come by default but don't. Most are inspired by other languages.
 
- String.atob()
- String.btoa()
- String.capitalize()
- String.chomp(chr)
- String.escapeHTML()
- String.md5()
- String.reverse()
- String.sha1()
- String.sha256()
- String.squeeze()
- String.times()
- String.trim()
- String.truncate()
